Details

Up 
get /place/details
Get details for a place (placeDetailsGet)
The API gets the details for a place. The place is identified by its reference. The place reference is available from the thindstodo API.

Query parameters

ref (required)
Query Parameter — Place reference string for the sight

Return type

PlaceDetails

Example data

Content-Type: application/json
{
  "country" : "aeiou",
  "country_code" : "aeiou",
  "offset" : 1.3579000000000001069366817318950779736042022705078125,
  "city" : "aeiou",
  "location" : {
    "lng" : 1.3579000000000001069366817318950779736042022705078125,
    "lat" : 1.3579000000000001069366817318950779736042022705078125
  },
  "state" : "aeiou",
  "state_code" : "aeiou",
  "url" : "aeiou"
}

Produces

This API call produces the following media types according to the Accept request header; the media type will be conveyed by the Content-Type response header.
  • application/json

Responses

200

An object with details for the place PlaceDetails

0

Unexpected error Error

get /place/events
Get the events for a place (placeEventsGet)
The API gets the events happening in the place of interest.

Query parameters

city (required)
Query Parameter — Name of the city

Return type

Events

Example data

Content-Type: application/json
{
  "events" : [ {
    "venue" : {
      "address" : "aeiou",
      "name" : "aeiou",
      "url" : "aeiou"
    },
    "city" : "aeiou",
    "title" : "aeiou",
    "url" : "aeiou"
  } ]
}

Produces

This API call produces the following media types according to the Accept request header; the media type will be conveyed by the Content-Type response header.
  • application/json

Responses

200

An object with events happening in the place Events

0

Unexpected error Error

get /place/photos
Get the URL for a photo (placePhotosGet)
The API gets the URL for the photo reference. The URL can be set as the src attribute for an img tag to see the photo for the reference.

Query parameters

photo_reference (required)
Query Parameter — Photo reference string for the sight

Return type

PhotoSrc

Example data

Content-Type: application/json
{
  "src" : "aeiou"
}

Produces

This API call produces the following media types according to the Accept request header; the media type will be conveyed by the Content-Type response header.
  • application/json

Responses

200

An object with src of the photo PhotoSrc

0

Unexpected error Error

Sights

Up 
get /place/thingstodo
List of sights to see for a given place (placeThingstodoGet)
The thingstodo endpoint returns information about the sights to see for a given place name. The name can be a city name or to get more specific results, one can provide the city, state, country name also.

Query parameters

place (required)
Query Parameter — Name of the place

Return type

Sights

Example data

Content-Type: application/json
{
  "place" : "aeiou",
  "list" : [ {
    "formatted_address" : "aeiou",
    "name" : "aeiou",
    "geometry" : "{}",
    "photos" : [ {
      "photo_reference" : "aeiou"
    } ]
  } ]
}

Produces

This API call produces the following media types according to the Accept request header; the media type will be conveyed by the Content-Type response header.
  • application/json

Responses

200

An object containing the place and list of sights to visit Sights

0

Unexpected error Error

Get the weather for a place (placeWeatherGet)
The API gets the weather for a place. The place can be the name of a city of a complete address of a place.

Return type

Weather

Produces

  • application/json

Responses

200

An object with weather of the place Weather
